# GreenCity Events Page Testing

## 📌 Project Description
This project contains automated UI tests for the GreenCity website.  
The tests are implemented using **Python**, **Selenium WebDriver**, and **unittest**.  
It covers functionalities like filtering events by date, searching events by keyword, and validating user input (email) on the login form.

## 🔗 Test Page
You can access the testing page here: [GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events)

## ⚙️ How to Run Tests
1. Make sure you have Python and Selenium installed.
2. Open a terminal in the project directory.
3. Run all tests with the following command:

```bash
python -m unittest discover tests

This will automatically discover and run all test scripts inside the tests folder.
